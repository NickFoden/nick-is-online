Projects Section

  • Salt and Pepper

    First API project I made, where you enter an ingredient and it queries a recipe API and displays a small card of a recipe you can make. It grabs 5 results and shows you a random recipe.

  • Cards

    A flashcards app to learn more javascript methods. Ended up learning PUG template engine to handle the views of this app. This project really became about using Passport JS for auth and how to set local session variables to mark the user as logged in. If you log in you can add new cards.

  • Proofer

    Upload an album of photos with one drag and drop. Pick the ones you approve of. Save as an album and then invite guests to your album and arrive at a consensus for which photos everyone can agree are publishable.
